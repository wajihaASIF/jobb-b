# Job Board

This is a user login, registration comments contact page role app using Node.js, Express, Passport, Mongoose, EJS and some other packages.

### Version: 2.0.0

### Usage

```sh
$ npm install
$ npm install express
```

```sh

# Or run with Nodemon

$ node app
$ npm start


# Visit http://localhost:5000
```
### Heroku
```sh
$ https://jobb-board.herokuapp.com/

```
